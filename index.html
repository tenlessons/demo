Working on the testing branch.

Commands:

  • git status - Shows you the current status of the repository
  • git log - Shows you the commit history for the repository
  • git add - Stages a file by filename or stages all files if you use the -A flag
  • git branch - Shows you all branches that currently exist.
  • git checkout - Switches to a the specified branch name or creates AND switches to the branch name if the -b flag is provided
  • git rm --cached - Unstage a file by filename or unstages all files using the -A flag
  • git commit - Commits all staged files use the -m flag to provide a commit message
  • git reset --soft HEAD~1 - Undo previous commit and keep files, if you want delete all changes use the --hard flag instead
